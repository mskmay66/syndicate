# Welcome to Syndicate

Your AI-powered, personal trading team. On this site you will find detailed installation and setup instructions as well as dev documentation.

## Getting Started

To install the package simply use python's built in package manager, pip:

```bash
pip install syndicate
```

Or, alternatively to build from the release:

```bash
pip install git+https://github.com/mskmay66/syndicate@alpha
```

Once installed there are two commands available to you. To enter the Terminal User Interface (TUI) run:

```bash
syndicate
```

If you already have an agent setup and want to force it to run you can call:

```bash
syndicate-run
```
