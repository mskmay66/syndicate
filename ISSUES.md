# Reporting Issues

If you encounter a bug, have a question, or want to request a feature, please report it through our GitLab issue tracker.

## 🐛 Where to Report

Submit all issues here:
👉 https://gitlab.com/your-username/syndicate/-/issues

## 📝 What to Include

To help us resolve issues quickly, please include:

* Clear description of the problem
* Steps to reproduce (if applicable)
* Expected vs actual behavior
* Logs or screenshots (if relevant)
* Environment details:
* OS (macOS, Linux, Windows)
* Python version
* Syndicate version

## 💡 Feature Requests

We welcome ideas for new features or improvements. Please:

* Describe the use case
* Explain why it would be useful

## 📌 Guidelines
* Search existing issues before opening a new one
* Keep reports concise and focused
* Be respectful and constructive

Thanks for helping improve Syndicate!
